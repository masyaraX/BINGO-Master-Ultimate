# BINGO Master Ultimate

学校行事、企業イベント、展示会、配信イベント向けのサーバー不要BINGOルーレットです。ブラウザだけで動き、LocalStorageに抽選履歴・番号データ・設定を自動保存します。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「開始」でルーレットを回し、「停止」で番号を確定します。
3. 番号をクリックすると、用語・説明・画像・URL・メモなどの詳細を確認できます。
4. 「データ管理」から番号データを編集し、JSONまたはCSVで入出力できます。

## 主な機能

- 1から75までのB/I/N/G/O盤、最大150番まで拡張可能
- 重複なしランダム抽選、任意設定で再抽選許可
- 自動抽選、高速抽選、一括抽選
- 抽選済み番号の発光表示、紙吹雪風パーティクル、効果音、簡易BGM
- 司会者モード、参加者モード、タイマー、景品メモ
- ライト、ダーク、ネオンテーマ
- JSON/CSVインポート・エクスポート
- PWA用manifestとService Worker

## CSV形式

ヘッダー行は次の列名に対応しています。

```csv
number,term,description,image,icon,category,url,memo,color,tags
```

`tags` は `|` 区切りです。

## オフライン利用

通常のファイル起動でも動作します。PWAのオフラインキャッシュを使う場合は、ローカルHTTPサーバーなどから開いてください。
